# 01_CS_Algorithm

## 시리즈 목적
자료구조와 알고리즘 학습 및 문제 풀이 기록을 위한 공간입니다.

## 학습 로드맵 (작성 예정 순서)
1. **시간 복잡도 (Time Complexity)** - 알고리즘 효율성 분석의 기초
2. **스택 & 큐 (Stack & Queue)** - 선형 자료구조의 이해
3. **기본 정렬** - 버블, 선택, 삽입 정렬의 구현과 특징
4. **이진 탐색 (Binary Search)** - 정렬된 데이터에서의 효율적 탐색
5. **트리와 그래프 (Tree & Graph)** - 비선형 자료구조의 기초
6. **DFS & BFS** - 그래프 탐색 알고리즘
7. **고급 정렬** - 퀵, 병합 정렬과 분할 정복
8. **엔트로피 (Entropy)** - 정보 이론 기초

## 작성 가이드
- 각 문제 풀이는 별도의 마크다운 파일로 작성합니다.
- 문제 설명, 풀이 전략, 코드 구현, 시간 복잡도 및 공간 복잡도 분석을 포함합니다.
- 예시: 백준 1000번 A+B 문제 풀이
