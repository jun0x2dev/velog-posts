# Velog Workspace

이 저장소는 jun0x2dev의 Velog 게시글 작성을 위한 워크스페이스입니다.
다양한 IT 기술 스택, 개인 프로젝트 경험, 그리고 학술적인 탐구를 기록하고 공유하기 위한 목적으로 구성되었습니다.

## 디렉토리 구조 및 시리즈 설명

- **01_CS_Algorithm**: 컴퓨터 공학(CS) 기초와 알고리즘 문제 해결 과정을 다룹니다.
- **02_Back-end**: Java와 Spring Framework를 중심으로 백엔드 개발의 깊이 있는 기술들을 탐구합니다.
- **03_Front-end**: React, TypeScript 등 최신 프론트엔드 생태계와 사용자 경험(UX) 개선 사례를 기록합니다.
- **04_Database**: 데이터베이스 모델링, 성능 최적화(Index), 트랜잭션 등 데이터 관리 기술을 다룹니다.
- **05_Infra**: 아키텍처 설계(Monolith vs MSA), CI/CD, 인프라 구축 및 운영 경험을 공유합니다.
- **06_AI**: 인공지능 관련 학습 내용과 기술적 탐구를 기록합니다.
- **08_Graduate_School**: 대학원 생활 중 진행한 연구, 학술 기록 및 학문적 고찰을 담습니다.
- **09_Personal_Project**: 개인 프로젝트의 시작부터 끝까지, 그 과정에서의 고민과 트러블슈팅을 정리합니다.
- **draft**: 아직 완성되지 않은 포스팅 초안들이 머무르는 공간입니다.
